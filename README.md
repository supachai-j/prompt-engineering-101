# Prompt Engineering 101

> Vendor-neutral fundamentals + production patterns for prompting any LLM

A bilingual (English + Thai) self-paced engineering training. Two tracks:
1. **Foundations** — for knowledge workers and beginners
2. **Production patterns** — for engineers building LLM applications

🌐 **Live:** https://supachai-j.github.io/prompt-engineering-101/ _(coming soon)_

## What's inside

- `index.html` — landing page
- `course-en.html` / `course-th.html` — self-paced courses (15+ modules)
- `slides/` — reveal.js decks for instructor-led sessions
- `wiki/` — cited knowledge pages (source of truth)
- `raw/` — captured primary sources

## Build

No build step. Open `index.html` in a browser, or push to GitHub Pages.
